# Lancer l'application

Ce document explique comment installer les dépendances nécessaires et lancer l'application à l'aide d'un script Bash.

## Prérequis

- Un système basé sur Debian ou Ubuntu
- Un terminal pour exécuter les commandes

## Étapes pour lancer l'application

1. **Executez le script d'instalation et de démarage**

   ```bash
   sudo chmod +x install_python.sh
   sudo ./install_python.sh
   ```

2. **Si déja installé, éxecutez les commandes suivantes pour lancer l'application.**

   ```bash
   cd minitel_project
   python main.py
   ```
