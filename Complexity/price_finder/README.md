Le code est composé d'une boucle while où à chaque itération, la taille de l'array est réduite de moitié.
La taille de l'array diminue exponentiellement à chaque itération.
Par conséquent, le nombre total d'itérations dans la boucle while est logarithmique par rapport à la taille de l'intervalle de recherche.
Ainsi, la complexité totale de l'algorithme est O(log(n)).
