Le code est composé d'une boucle récursif :

- Pour chaque élément dans l'array, la fonction appelle récursivement la fonction en excluant le premier élément.
  Ainsi le nombre total d'appels récursifs est proportionnel au nombre d'éléments dans l'array, soit O(n).

Ainsi, la complexité totale de l'algorithme est O(n).
